Highlight #3 Domain Name System

The Domain Name System (DNS) is a system that organizes the domain names and translates IP addresses into these domain names. Websites on the internet can be reached using the IP address of the web server currently hosting the website. Because these IP addresses are not easily memorized by people, IP addresses are not a great way to humans to reach their sought after website. As a result, we use domain names to replace IP addresses. If a person types in the domain name of the website they are trying to reach, the DNS will find the matching IP address of that website allowing the user to reach the site. This topic interests me because it is fascinating how the internet works and organizes the millions of websites on the internet. The DNS system is a great way to make it easier for people to remember the websites they want to go to and organize websites in a way that makes sense. The Domain Name System is relevant to this course because the web applications that we create will be reachable by IP addresses but will also have a domain name based on the web server hosting the application.

Highlight #4 HTML

HTML is the language that is used to create web pages. HTML stands for HyperText Markup Language. Websites are coded using HTML by following the rules and formatting guidelines of the language. This language uses special markers to specify formatting and text properties. For instance, to bold text in HTML the code will include before the text and after the text. All computer systems that are available today understand HTML so no matter what computer you are on, the web page will look the same. HTML helps to ensure that these websites are compatible across all devices, whether that is phones, tablets, or computers. Furthermore, HTML makes sure that websites are accessible for people with disabilities such as seeing problems. This topic interests me because I really enjoyed creating websites in HTML last semester. Also, I find it very interests how HTML converts the users input into the visual representation of the website. HTML is relevant to this course because the websites we will be creating will be coding using HTML.

Highlight #5 CSS

CSS is a stylesheet language that is used to describe how HTML elements are displayed on a web page. CSS stands for Cascading Style Sheets. CSS is used to format and style HTML elements on a web page. With CSS, users can implement unique styles, colors, and texts to make the web page more distinctive and appealing. Furthermore, multiple pages of HTML can be formatted with one page of CSS by using an external stylesheet. This makes it very easy to style an entire website with a few external stylesheets in CSS. Also, without CSS, the uniqueness of web pages is very limited as the formatting and styles are restricted only using HTML. This topic interests me because I find it extremely fascinating how CSS can alter the look of web pages using simple commands. Another reason CSS is interesting is that it makes it very easy to make a web page look however the user wants it to look. CSS is relevant to this course because it is a very important part of web design so it will be integral in web application development.
